Lists

There are three types of list but I will be using only two types; ordered and unordered lists.

Ordered List

Ordered lists are marked up using ol tags. An ordered list may have various numbering schemes presented through CSS.

  1. list item one
  2. list item two
  3. list item three
  4. list item four
  5. list item five

Unordered List

Unordered lists are marked up using ul tags. An unordered list describe a collection of things.

  • list item one
  • list item two
  • list item three
  • list item four
  • list item five

Nested List

This is a list nested into another list.

  • list item one
    1. list item two
    2. list item three
  • list item four
  • list item five
